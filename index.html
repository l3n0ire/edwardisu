Questionbank Contributions

  1. The negative Xi baryon (Ξ-) has the quark composition of dss.
    1. State its charge (Q), lepton number (L), and baryon number (B).
    2. Hence, determine whether or not it can decay into a neutral Lambda baryon (Λ0, uds) and a negative pion (π-, du).
    3. Can the decay products be observed in a bubble chamber? Explain.


  2. Consider the following four particles:
    • uud
    • ud
    • μ
    • γ

    1. Which of the above particles are hadrons?
    2. Which of the above particles have a charge of -1e?
    3. Which of the above particles have a half-integer spin?
    4. Which of the above particles have an integer spin?

Solutions

    1. Q = -1
      L = 0
      B = 1
    2. Q = 0, -1 = -1
      L = 0, 0 = 0
      B = 1, 0 = 1
    3. The bubble chamber can only detect charged particles. Therefore, the negative pion can be observed, while the neutral Lambda baryon cannot.


    1. uud, ud
    2. μ
    3. uud, μ
    4. ud, γ
